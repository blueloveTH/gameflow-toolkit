# GameFlow工具包（测试版）

为同时兼任策划和程序的游戏制作人打造的开发工具包！

A Lightweight, Designer-Oriented Toolkit for Unity Games.

<br>

**GameFlow-Tookit（简称gf）** 是一款用于制作Unity游戏的开源工具包，出自2018年秋季探索（Autumn Quest），作为GameDevKit核心组件，经过多个独立项目的实践积累，精化并逐步完善。特点如下：

+ 快速实验玩法
+ 轻量、极简的API
+ 初级到进阶适用
