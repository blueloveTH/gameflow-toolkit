# GameFlow工具包（测试版）

**GameFlow-Tookit（简称gf）** 是一款用于制作Unity游戏的开源工具包。它为独立开发者，尤其是同时兼任程序和策划的制作人而设计。通过提供线性和非线性的图结构以及常用设计模式，使游戏设计师可以快速构建可扩展的游戏机制。



> gf原先是一个名为GameDevKit项目的一部分，出自2018年秋季探索（Autumn Quest），精化并独立出来。经过多个项目实践经验的积累，逐渐得到完善。
